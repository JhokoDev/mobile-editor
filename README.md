# Code Studio Mobile

Um editor de código leve e moderno, construído com **Vite**, **React** e **TypeScript**, focado em produtividade móvel.

## 🚀 Funcionalidades
- **Suporte Multi-Linguagem**: Realce de sintaxe para Python, JavaScript, CSS, JSON e mais.
- **Estrutura Vite**: Performance ultra-rápida e hot reload otimizado.
- **Design Mobile-First**: Interface pensada para telas pequenas, com menus deslizantes e áreas de toque otimizadas.
- **Explorador de Arquivos**: Navegação intuitiva por pastas e arquivos.
- **Barra de Status Dinâmica**: Informações em tempo real sobre o ambiente de codificação.

## 🛠️ Como Utilizar
1. **Abrir Arquivos**: Use o menu lateral (ícone de hambúrguer) para navegar pelos arquivos.
2. **Editar**: O editor central exibe o código com realce de sintaxe baseado na extensão do arquivo.
3. **Executar**: O botão "Run" simula a execução do ambiente (preparado para integrações futuras).

## 💻 Tecnologias
- **Vite** (Build Tool)
- **React 19**
- **Tailwind CSS v4**
- **Motion** (Animações)
- **Lucide React** (Ícones)
- **React Syntax Highlighter** (Prism)
