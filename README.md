# Mobile Code Studio

Interface de editor de código inspirada no VS Code, totalmente otimizada para dispositivos móveis.

## 🚀 Como Utilizar

### 1. Navegação (Activity Bar)
Utilize a barra lateral esquerda (ou inferior em alguns contextos) para alternar entre as ferramentas:
- **Explorer (Arquivos)**: Navegue pela estrutura de pastas e arquivos do projeto.
- **Search (Busca)**: Realize pesquisas rápidas no código.

### 2. Explorador de Arquivos
- Clique em qualquer arquivo para visualizá-lo no editor central.
- No **Mobile**, utilize o ícone de menu (hambúrguer) no topo para abrir/fechar o explorador.

### 3. Editor de Código
- O editor central exibe o código com realce de sintaxe automático.
- Suporta rolagem horizontal e vertical suave.
- Otimizado para toque, com espaçamento adequado para o teclado virtual.

### 4. Barra de Status
- Localizada no rodapé, exibe informações como a branch atual (`main`), status do Prettier e metadados do arquivo (linha/coluna).

## 🛠️ Tecnologias Utilizadas
- **Framework**: Next.js 15 (App Router)
- **Estilização**: Tailwind CSS v4
- **Animações**: Motion (framer-motion)
- **Ícones**: Lucide React
- **Syntax Highlighting**: React Syntax Highlighter (Prism)
